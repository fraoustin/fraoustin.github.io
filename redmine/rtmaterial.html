Summary
RTMaterial is a theme for redmine.


The theme use two colors: Primary and Accent.
List of color:
  • red
  • pink
  • purple
  • deepPurple
  • indigo
  • blue
  • lightBlue
  • cyan
  • teal
  • green
  • lightGreen
  • lime
  • yellow
  • amber
  • orange
  • deepOrange
  • brown
  • grey
  • blueGrey
You can add a specific color theme by project: You must use redmine_rtmaterial plugin. The redmine_rtmaterial plugin add field "RTMaterial" for project. You can selected color for RTMaterial Theme by project Or you can selected color default for RTMaterial Theme by project.
Installation
For the theme 
cd /usr/src/redmine/public/themes
git clone https://github.com/fraoustin/RTMaterial.git
For redmine_rtmaterial plugin 
cd redmine/plugins
git clone https://github.com/fraoustin/redmine_rtmaterial.git
rake redmine:plugins:migrate NAME=redmine_rtmaterial
and restart redmine
