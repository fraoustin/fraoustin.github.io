Flow
A flow can is downloaded and uploaded.
A flow has
  • name
  • resume
  • environment
  • logs
A environment management give by environment value of key
You can use key with 
<< key1 >>
the key default of environment:
  • _id: id of flow
  • _name: name of flow
  • _version
  • _levellog: level log of run
You can test environment with this flow
You can run a flow with run button with level log and environment.

You can save flow with version notion.
