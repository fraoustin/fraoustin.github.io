Installation
Bambou use Python >= 3.7 
git clone https://github.com/fraoustin/bambou.git
cd bambou
pip install -r REQUIREMENTS.txt
python bambou.py
You can use bambou on http://127.0.0.1:5000/ with
  • login admin@localhost
  • and password admin
Configuration
You can use the environment for configuration of bambou
  • BAMBOU_DIR: path of bambou
  • BAMBOU_PORT: the port of the webserver. Defaults to 5000
  • BAMBOU_HOST: the hostname to listen on. Defaults to 0.0.0.0
  • BAMBOU_LOGS: directory of log
You can use bambou.cfg 
[BAMBOU]
Host = 0.0.0.0
Port = 5000
Debug = false
# Db =
# Logs =
Docker
You can build image and use 
git clone https://github.com/fraoustin/bambou.git
cd bambou
docker build -t bambou .
docker run -p 5000:5000 -d bambou
You can use bambou on http://127.0.0.1:5000/ with
  • login admin@localhost
  • and password admin
New from 0.8.1
You can use a image from dockerhub 
docker run -p 5000:5000 -d fraoustin/bambou
You can add directory for database and data 
docker run -p 5000:5000 -d -v <your-path>:/db -v <yourpath>:/data fraoustin/bambou
Windows
You can download and unzip the package https://github.com/fraoustin/bambou/releases/download/0.9.1/bambou_0.9.1.zip. You can load bamabou.exe and you can use bambou on http://127.0.0.1:5000/ with
  • login admin@localhost
  • and password admin
