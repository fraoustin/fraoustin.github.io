General
A node has
  • main input (in1)
  • optional input (in2)
  • main output
  • error output
You can test multi nodes this flow
Synchronize
In
  • in1
  • in2
Out
  • in1
  • error
You can use shortcurt
  • Crtl+Alt+c : copy node
  • Crtl+Alt+v : paste node
  • Crtl+s: save node
  • Crtl+entr: run flow
  • Crtl++ zoom in
  • Crtl+- zoom out
  • Shift mousewheel zoom in or out
